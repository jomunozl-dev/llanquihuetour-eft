# llanquihuetour-eft
Aplicación desarrollada en Java como Examen Final Transversal de Fundamentos de Programación Orientada a Objetos. Simula la gestión de la agencia Llanquihue Tour, aplicando encapsulamiento, herencia, polimorfismo, interfaces, composición, modularidad, colecciones, lectura de archivos y validación mediante excepciones personalizadas.
